    QuizzTp2¶ ↑

    Trabalho realizado para a disciplina Técnicas de Programação 2, visando praticar técnicas de teste e modularização.

    Gabriel Rodrigues Diógenes Macêdo - 15/0126808 Cristiane Naves Cardoso - 15/0008023 Rômulo de Vasconcelos Feijão Filho - 140031260

    -A primeira tela é a tela de login. O usuário pode logar com seu login ou criar sua própria conta. -Existem dois tipos de conta, as normais e as contas admin -Apenas contas admin podem criar quizes e disciplinas -Os quizes podem ser no formato V e F ou multipla escolha -Foi adicionado o caderno do usuario contendo todas as disciplinas em que ele esta cadastrado -Tambem foi adicionado a lista geral de disciplinas -Foi adicionando um score para o usuario saber quantos pontos ele fez em algum quiz
